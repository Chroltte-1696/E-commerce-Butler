# E-commerce-Butler
我构建了一个基于AI Agent的电商多平台商品同步管家，核心解决淘宝千牛与微信视频号小店之间“一货多上架”的高重复人力痛点。传统模式下，运营需在两个平台反复填录标题、属性、尺码表、详情图与价格策略，单SKU上架耗时超15分钟且极易出错。

该工具采用多Agent协作架构：主调度Agent接收“以千牛某商品为母版”指令后，唤醒平台适配Agent群。千牛解析Agent抓取源商品全量信息并结构化输出；智能转换Agent执行长链推理——自动完成类目映射（如千牛“连衣裙”映射到视频号“女装-连衣裙”）、文案风格改写（适配微信私域语境）、图片合规裁剪（规避视频号限高文字密度要求）及差异化定价计算。生成预览稿后，视频号上架Agent执行最终发布并回流新商品ID，全程耗时压缩至30秒，准确率达98%，彻底释放了运营人力。
